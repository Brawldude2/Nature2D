Nature2D is a vast 2D physics library designed for and on [Roblox](https://www.roblox.com/)! Ever wanted to create 2D games but step back because Roblox doesn't have a 2D Physics engine? Use Nature2D to create versatile and smooth simulations and mechanics for your 2D games with minimum effort!
